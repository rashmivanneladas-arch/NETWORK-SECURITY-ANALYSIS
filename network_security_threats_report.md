# Network Security Threats Report

## Introduction
Network security protects systems and data from cyber attacks.

## DoS Attack
A Denial of Service attack overloads a system and makes it unavailable.

## Man-in-the-Middle Attack
The attacker secretly intercepts communication between two parties.

## Spoofing Attack
Spoofing is pretending to be a trusted source to gain access.

## Prevention Methods
- Use firewalls
- Use encryption
- Update systems regularly
- Use strong authentication

## Conclusion
Understanding threats helps improve cybersecurity and network protection.
